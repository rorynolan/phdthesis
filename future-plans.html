Algorithms for the Correction of Photobleaching

Chapter 6 Future plans

6.1 Robin Hood publication

The Robin Hood algorithm is already incorporated in the R package detrendr but it hasn’t been published or peer-reviewed. Getting this done is my top priority.

6.2 Translate software to ImageJ

Whilst I really like R, the fluorescence community does not use it, which has been a major barrier to the use of my algorithms. I will code my detrending algorithms as ImageJ plugins (and perhaps also as python modules) so that more of the community have easy access to them.
